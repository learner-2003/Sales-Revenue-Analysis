## Sales & Revenue Analysis Dashboard

## Overview

The Sales & Revenue Analysis Dashboard is an interactive business intelligence application developed using Python, Streamlit, Pandas, and Plotly. The dashboard helps organizations analyze sales performance, monitor key business metrics, identify trends, and generate actionable insights through data visualization.

This project demonstrates data analysis, KPI tracking, dashboard development, and business insight generation using a real-world sales dataset.


## Features

### 1. KPI Tracking

* Total Revenue
* Total Profit
* Total Orders
* Total Quantity Sold

### 2. Interactive Filters

* Region Filter
* Category Filter
* Sales Channel Filter

### 3. Data Visualization

* Revenue Trend Analysis
* Profit Trend Analysis
* Top 10 Performing Products
* Revenue Distribution by Region
* Category Performance Analysis

### 4. Business Insights

* Best Performing Product Identification
* Best Revenue Generating Region
* Sales Improvement Recommendations

### 5. Data Management

* Interactive Sales Records Table
* Download Filtered Data as CSV

bash

bash

### Run the Application

bash
